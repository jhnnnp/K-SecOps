시나리오 데모 허브

시나리오마다 흐름이 다릅니다. PR (5·6)은 코드 변경 → commit → push → PR → GitHub CI → 대시보드입니다. Local (2·3)은 내 PC에서만 gate를 돌리며 GitHub push/PR 없음.

어떤 버튼이 commit · push · PR 을 하나?

  • Scenario 5 · PR PASS — dummy-infra/README.md 변경 → commit → push → PR → CI 초록
  • Scenario 6 · PR FAIL — src/main.py 예시 키 추가 → commit → push → PR → CI 빨강 · merge 차단
  • Scenario 2·3 — 로컬 ci_gate.py 만 실행 (commit/push/PR 없음, 빠른 UI 확인용)

잘못된 접속 방식

HTML 파일 직접 열기(file://) 또는 GitHub Pages에서는 API가 동작하지 않습니다.

python3 scripts/demo_hub.py → http://127.0.0.1:8765
